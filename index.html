CSS Snippet Cheatsheet

Handy CSS tools for when you just can't remember the code!

Media Query

You can use this Media Query to change the flex display depending on screen size.

@media screen and (max-width: 768px) {
    section {
        width: 90%;
    }
}

Pointer finger on link hover

This turns your cursor into a hand with a pointed finger when hovering over a link.

a:hover {
    cursor: pointer;
}

General Attribute Value Selector

Change "class" to a specified class and "val" to the value you are searching for.

[class*="val"] {
    font-size: 20px;
}

Root Variable for Text Shadow

This is a simple root variable (--gtext) that gives text a white glow.

:root {
    --gtext: 3px 0px 5px white
}

Box Transition Glow

This adds a green glow effect when you hover over an item. Change the color to make it your own!


section:hover {
    border-radius: 5px;
    box-shadow: 0 0 20px 10px chartreuse;
    transition: all 0.5s ease-in-out;
}

Flexbox Row w/ Wrap using Flex Flow

This allows you to define a wrapping flex row with just one line!

div {
    flex-flow: row wrap;
}

CSS makes everything better!
