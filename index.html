CSS Snippet Cheatsheet

Handy CSS tools for when you just can't remember the code

Media Query

You can use this Media Query to change the flex display depending on screen size.

@media only screen and (max-width: 600px) {
    flex-wrap: wrap;
    flex-direction: column;
}

Pointer finger on link hover

This turns your cursor into a hand with a pointed finger when hovering over a link.

a:hover {
    cursor: pointer;
}

General Attribute Value Selector

You can use this to apply css properties to any specified content. This example would change the font size of every class containing the letters "me".

[class*="me"] {
    font-size: 20px;
}

Fixed Header

This is a simple fixed header that sticks when you scroll.

header {
    background-color: #fff;
    position: fixed;
    top: 0;
    width: 100%;
}

Box Transition Glow

This adds a glow effect when you hover over an item.

.code-card .card-header {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
}
              
.code-card:hover, .code-card:hover .card-header {
    box-shadow: inset 0px 0px 8px rgba(232, 102, 236, 1), 0 0 15px rgba(232, 102, 236, 1);
}

Flexbox Row w/ Wrap using Flex Flow

This allows you to define a flex row with just one line!

div {
    flex-flow: row wrap;
}

CSS makes everything better!
